# QA Project – OrangeHRM My Info Module

## 📌 Project Overview
This project was developed as a final assignment in a Software Testing course.  
It focuses on analyzing requirements and designing structured test cases for the *My Info* module of OrangeHRM.

## 🎯 Objectives
- Derive testable requirements from an SRS document
- Design structured test cases
- Ensure full traceability
- Analyze quality metrics (KPIs)

## 📊 Project Metrics
- 26 Requirements
- 293 Test Cases
- 100% Requirement Coverage
- Functional & Non-Functional Testing
- Formal Test Design Techniques applied

## 🧪 Techniques Used
- Use Case Testing
- Equivalence Partitioning
- Boundary Value Analysis
- Decision Table Testing
- State Transition Testing

## 📁 Contents
- Test Cases
- Traceability Matrix
- KPI Analysis
- Final Report

## 🛠 Tools Used
- Microsoft Excel
- Microsoft PowerPoint
- Structured QA methodology (ISTQB-aligned)
